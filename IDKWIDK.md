# IDKWIDK Protocol

## The 7 Gates

1. **What We Built** - Complete inventory
2. **What We Didn't Build** - Missing scope
3. **What Will Break** - First failure mode
4. **What You Can't See** - Domain blind spots (MUST surface 1+ unasked thing)
5. **What You'll Forget** - Resumption context (3 days, 3 weeks)
6. **The Simpler Version** - 80/20 check
7. **Do This Next** - 1-3 actions, each under 30 minutes

## Rules

- Always run. "Outside scope" = reason to run, not skip.
- Silence = failure. Every gate must have content.
- Gate 4 must surface at least 1 thing the user didn't ask about.
- Gate 7 actions feed into the action plan via `idkwidk-action-plan.py`.
